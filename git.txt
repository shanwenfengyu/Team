将项目通过Git来进行管理
1.自己创建好的基本项目结构
2.将我们的项目变成一个Git仓库->git init
3.因为我们代码仓库只需要放入我们的代码,就像class文件,编译器自身生成的文件
文件都不需要放入到仓库中,而且如果还会经常出现代码冲突,解决方案就是使用
忽略文件进行设置.gitignore
4.在设置好忽略文件后我, 就可以将我们的代码提交到我们的本地仓库了.
这时候对于我们的文件我们需要先通过git add 要添加的文件的路径   来对我们要放入本地仓库的代码
进行添加操作,接下来通过git commit -m "提交信息"来真的将我们的代码提交到本地仓库,
这时候才是真的将代码保存在 本地仓库了,我们希望跟别人共同进代码的管理就需要将我们
本地的仓库和远程git仓库进行连接并将代码推送到远程服务器上
这样才能和别人共同开发git remote add origin 远程服务器上面的仓库地址
这样连接好之后我们就可以进行项目从本地到服务器的推送了
git push -u origin master
